Features

Built for how developers

No subscription, no telemetry, no fluff. Just a solid Git GUI that covers your daily workflow.

🌿

Visual branch graph

Keep track of your full project history at a glance — merges, rebases, and divergences rendered clearly.

⚡

Monaco diff viewer

Side-by-side and inline diffs with syntax highlighting for every language.

🎯

Interactive staging

Stage individual hunks or lines. Craft precise, atomic commits without touching the command line.

🔄

Rebase & stash

Interactive rebase flow, stash management, and conflict resolution — all from the same interface.

✨

Clean history

Fixup and squash commits with one click. Reword messages, reorder, drop — keep your history clean before it hits the remote.

🏷️

Tags, remotes & more

Push, pull, fetch, create and delete branches, manage remote refs, and tag releases — everything in one place.

Multi-lane branch visualization with color-coded lanes, author avatars, and instant filtering.
